# apod-grabber
Utility to set the Mac desktop to the current NASA astronomy photo of the day, including explanation text overlaid onto image.

Usage:
 python3 apodgrab.py

Requirements:
 - BeautifulSoup (pip3 install beautifulsoup4)
 - Pillow (pip3 install pillow)
